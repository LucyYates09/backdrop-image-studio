# Backdrop 图片背景工作台

一个纯前端、无需登录的图片背景处理工具。适合给产品截图、演示图片和社交媒体素材添加渐变背景、留白、圆角、阴影与品牌水印。

## 在线使用

仓库启用 GitHub Pages 后，可直接通过公开网页使用。打开页面后，将图片复制到剪贴板，再按 `Command + V`（macOS）或 `Ctrl + V`（Windows/Linux）粘贴即可。

## 主要功能

- 直接从剪贴板粘贴图片，无需上传文件
- 支持 `3:4`、`2:3`、`4:5`、`9:16`、`1:1`、`4:3`、`3:2`、`1.91:1`、`16:9` 等常用比例，覆盖 Instagram 方形、竖版、横版和 Story / Reel
- 多种渐变和纯色背景，并支持自定义颜色
- 可调节图片外边距、圆角和阴影
- 可选品牌水印，默认文字为 `Power by JobWizard.ai`
- 水印文字可编辑，颜色支持自动、黑色和白色
- 支持无损 PNG 复制和下载
- 尽量保留原图像素，超大画布设有安全上限

## 隐私说明

所有图片处理都在浏览器本地完成。图片不会上传到服务器，也不会保存到 GitHub。

## 本地使用

下载仓库后，直接双击 `index.html`，或将该文件拖入浏览器即可使用，无需安装任何依赖。

## 技术说明

项目为单文件静态网页，仅使用原生 HTML、CSS、JavaScript 和 Canvas API，没有后端服务和第三方运行时依赖。

## 发布方式

本项目通过 GitHub Pages 从 `main` 分支根目录发布。更新 `index.html` 并推送到 `main` 后，线上页面会自动更新。
